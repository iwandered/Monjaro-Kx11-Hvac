# Monjaro-Kx11-Hvac
2021-Monjaro-4Rhythm-Kx11-Hvac
星越L空调、座椅加热控制面板
基于DeepSeek，通义，Gemini等AI
已验证车型：2021款吉利星越L四驱旗舰型，车机型号Kx11
编译需在app/libs中放入ecarx库，本代码未提供
目前无法实现风量控制、设定温度显示和控制以及与风量控制和温度控制相关的功能。
目前实现主副驾驶座椅加热，风向一键调节和组合调节，AC开关，AUTO开关，内外循环切换，空气净化g-clean一键开关，车内外温度传感器显示，手动、自动风量显示，副驾驶有人时显示欢迎文字，点击左下角Home键返回默认桌面，点击hvac打开原车空调，点击seat打开原车座椅设置界面
